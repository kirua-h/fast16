# Fast16

Fast16は16時間断食を管理するためのシンプルなiPhoneアプリです。

## MVP機能

- 食事開始時刻の記録
- 残り時間のリアルタイム表示
- 次に食事できる時刻の表示
- ローカル保存
- ダークモード対応

## 使用技術

- Expo
- React Native
- TypeScript

## 開発

依存関係をインストール

```
npm install
```

開発サーバー起動

```
npx expo start
```
