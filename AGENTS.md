# Fast16 AI開発ガイド

## プロジェクト概要

Fast16は16時間断食をサポートするiPhoneアプリです。

目的は「食事を開始した時刻を記録し、次に食事ができるまでの残り時間をシンプルに確認できること」です。

アプリはできる限りシンプルに保ち、必要以上の機能は追加しません。

---

## 開発方針

以下を優先してください。

1. シンプルであること
2. 安定して動作すること
3. 読みやすいコードであること
4. Apple純正アプリのような体験であること

---

## 使用技術

- Expo
- React Native
- TypeScript

---

## UIルール

Apple純正アプリのようなミニマルなデザインを採用します。

- 余白を広く取る
- 色は最小限にする
- 派手なアニメーションは追加しない
- 残り時間を画面で最も目立たせる
- 情報量は必要最低限にする

---

## 開発ルール

- 1機能ずつ実装する
- 常にアプリが起動する状態を維持する
- 勝手に仕様を変更しない
- 不要なライブラリを追加しない
- コンポーネントは小さく保つ
- 重複コードを書かない
- TypeScriptでanyは使用しない
